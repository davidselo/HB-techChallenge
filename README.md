## `hb pick-run`

Get optimal pick run base on a csv file with product locations.

```
USAGE
  $ hb pick-run

FLAGS
  -h, --help     Show CLI help.
  -v, --verbose

DESCRIPTION
  Get optimal pick run base on a csv file with product locations.
```
